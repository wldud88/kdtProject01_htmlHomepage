맞춤형 관광지도 서비스

본 서비스는 사용자 맞춤형 데이터를 기반으로
여행자의 성향, 현재 위치, 테마 관심도 등을 분석하여
가장 적합한 관광지를 지도 위에 시각적으로 추천해주는 시스템입니다.

주요 기능 안내

  • 관광지 키워드 검색 기반 테마 자동 추출
  • 거리 기반 Top10, 인기 기반 Top10 관광지 추천
  • 성별/연령/재방문 의향 등 세부 필터
  • Leaflet 기반 지도 + 마커 + 상세 정보
  • FastAPI + MySQL 연동으로 실시간 응답

이 시스템은 공공 데이터 활용, 관광 활성화, 사용자 편의성을 최우선으로 기획되었습니다.
